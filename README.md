## 📖 What Are ASCII Art Banners?

> **ASCII Art** is the general term for art made entirely from text characters.  
> These stylized text banners you see in terminals and scripts are more specifically known as **FIGlet text** — named after the legendary tool that generates them.

| Term | Description |
|------|-------------|
| **ASCII Art** | Art made purely from standard text characters |
| **FIGlet Text** | Stylized text banners generated by the `figlet` tool |
| **Banner Text** | Another common name for large text headers |
| **Text Art Headers** | Decorative text used at the top of scripts or logs |

## 🛠️ How to Create Them

### 1️⃣ Using `figlet` — *The Classic CLI Tool*

> [!NOTE]
> `figlet` is the original and most widely-used ASCII banner generator.

**Install:**
```bash
sudo apt install figlet
```

**Basic usage:**
```bash
figlet "Hello World"
```

**Use a specific font:**
```bash
figlet -f slant "Hello World"
```

**Browse all available fonts:**
```bash
showfigfonts
```

### 2️⃣ Using `toilet` — *FIGlet with Colors!* 🎨

> [!TIP]
> `toilet` extends `figlet` with color support and special rendering filters — great for making your banners pop!

**Install:**
```bash
sudo apt install toilet
```

**Basic usage:**
```bash
toilet "Hello"
```

**Metallic effect:**
```bash
toilet -f mono12 -F metal "Hello"
```

**Other cool filters:**
```bash
toilet -f mono12 -F gay "Rainbow!"       # rainbow colors
toilet -f mono12 -F border "Boxed"       # add a border
```

### 4️⃣ Using in Your Bash Scripts 📜

Paste the generated art directly into your script using a **heredoc**:

```bash
cat << 'EOF'
  _   _      _ _       
 | | | | ___| | | ___  
 | |_| |/ _ \ | |/ _ \ 
 |  _  |  __/ | | (_) |
 |_| |_|\___|_|_|\___/ 
EOF
```

> [!IMPORTANT]
> Always use `<< 'EOF'` (with **quotes** around EOF) so that Bash doesn't try to interpret `\` and `$` characters inside the art as escape sequences or variables.

## 💡 Pro Tips

- 🎯 **Pick readable fonts** — `slant`, `banner`, `big`, and `standard` are great defaults
- 🎨 **Combine `toilet` with `lolcat`** for rainbow gradient output:
  ```bash
  figlet "Cool" | lolcat
  ```
- 📏 **Check terminal width** — some fonts produce very wide output; use `-w` flag to set max width:
  ```bash
  figlet -w 80 "Fits!"
  ```
- 🔤 **Preview fonts quickly** with `showfigfonts | less` to scroll through all installed fonts
